# Festive Premium — Shopify Theme for Creative Gifts Store

A high-end holiday-themed Shopify theme designed for creative gift stores. Warm, elegant, and conversion-focused — perfect for Christmas, Valentine's Day, birthday gifting, and year-round occasions.

---

## Theme Features

- **Festive Atmosphere** — Deep wine and dark gold palette with snowfall/starlight CSS animations
- **Holiday Countdown Module** — Urgency-building countdown timer for seasonal campaigns
- **Hero Banner with CTA** — Full-width gradient hero with animated particles
- **Featured Collection Grid** — Card-based product showcase with smooth hover effects
- **Testimonial Carousel** — Swipeable customer reviews for social proof
- **Mobile-First Responsive** — Looks flawless on desktop, tablet, and mobile
- **Theme Settings Panel** — Holiday mode toggle, colour pickers, and section controls via Shopify Customizer
- **SEO Optimised** — Clean semantic HTML, fast CSS-only animations, minimal JS footprint

---

## Quick Start — Connect to Shopify via GitHub

### Step 1: Log in to GitHub
From your Shopify Admin, go to **Online Store → Themes → Add Theme → Connect from GitHub**. Click **Log in to GitHub** and authorize Shopify to access your repositories.

### Step 2: Select This Repository
Once connected, select this repository from the dropdown. Shopify will sync the theme files automatically.

### Step 3: Publish & Customise
After syncing, click **Customize** to open the Theme Editor. Adjust colours, toggle the holiday countdown, upload your logo, and add your products — all without touching code.
